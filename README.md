# 微信小程序版本的俄罗斯方块

一个功能丰富的俄罗斯方块小游戏，支持多种游戏模式、联机对战、数据统计等特性。

## 功能特点

### 游戏模式
- 🎮 经典模式：无尽模式，速度逐渐加快
- 🏆 挑战模式：40行挑战，看谁用时最短
- ⏱️ 限时模式：2分钟内获得最高分
- 🧘‍♂️ 禅模式：固定速度，无压力享受
- 🏃 马拉松模式：坚持到150行
- ⚡ 冲刺模式：20行最快时间
- 解谜模式：特定形状挑战
- 🗑️ 垃圾回收：清理随机垃圾行
- 👁️ 盲打模式：隐藏下一个方块
- 🪞 镜像模式：操作反转

### 联机对战
- 实时多人对战
- 垃圾行系统
- 实时聊天
- 快捷消息
- 对战统计

### 数据分析
- 详细游戏统计
- 趋势图表
- 方块使用分析
- 游戏时长统计
- 数据导出功能

### 成就系统
- 多种成就类型
- 解锁进度追踪
- 成就点数统计
- 解锁提示动画

### 排行榜
- 总榜/每日榜/周榜
- 模式专属排行
- 成就点数排名
- 好友排名

### 其他特性
- 主题切换
- 自定义主题
- 游戏录像
- 手势控制
- 键盘支持
- 音效系统
- 数据云同步
